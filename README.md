### Wisconsin Real Estate Data Analysis

### Objective: 
Exploring and cleanning Wisconsin real estate sales data (2022) and preparing it for visualization and simple modeling. The notebook stages data into SQLite, cleans it, enforces a schema, and explores property trends across the state.

### Notebook overview

This notebook follows these main steps:
-Load the raw CSV into pandas
-Stage the data in a local SQLite database
-Clean the data (remove duplicates, fix missing values, correct datatypes)
-Enforce a tighter schema with clear column types
-Explore and visualize key trends in the dataset

### Tools used

-Python
-pandas – for data handling
-SQLite / SQLAlchemy – for database staging and queries
-matplotlib / seaborn – for charts and visualizations
